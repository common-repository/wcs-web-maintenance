Add feature "Under Construction" or "Site closed for maintenance"

== Description ==

WCS Web Maintenance is a **lightweight, elegant and powerful** solution that allows you to change a website to "Under Construction" or "Site closed for maintenance" while you can make changes to the content, design and configuration.

= Features =

*   Configure text message and color styles
*   For administrators, add a widget to "Dashboard"
*   Customize type of users for those who must restrict access to the site

Supported languages:

*   English (EN)
*   Spanish (ES)

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `wcs-web-maintenance` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the WCS Web Maintenance settings page and configure it
